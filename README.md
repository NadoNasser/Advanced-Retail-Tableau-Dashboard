# Advanced Retail Analytics & Supplier Operations Dashboard

An interactive, dual-dashboard Business Intelligence solution built using Tableau Desktop. This project bridges the gap between high-level retail performance tracking and granular supplier logistics, transforming a multi-layered retail dataset into actionable, decision-ready insights.

🔗 **Live Project Link:** https://public.tableau.com/app/profile/nada.eissa/viz/Nada_Dashboard/ExecutiveOverview?publish=yes


## 🚀 Advanced Features Implemented

* **Dynamic Metric & Dimension Toggles:** Built custom parameters allowing users to dynamically switch between *Sales* and *Profit* views across different grains (*Category* vs. *Region*).
* **Dual-Axis Continuous Timeline:** Mapped financial performance markers against transaction volume (`Quantity`) to isolate value growth vs. volume surges.
* **Dynamic Zone Visibility:** Engineered sheet-swapping rules within layout containers to toggle clean *Trend Views* and *Summary Tables* seamlessly.
* **Asymmetric Drill-Downs:** Configured advanced **Set Actions** allowing user clicks to dynamically expand category profiles to the item level without cluttering vertical dashboard real estate.
* **Cross-Dashboard Navigation & Action Filters:** Added custom navigation routes between dashboards and mapped strict action-filtering rules to expose supplier metadata and contact lines directly upon item selection.

---

## 📊 Dashboard Architecture

### 1. Executive Overview
Designed for leadership to monitor key financial indicators (Sales, Profit, Quantity, and Average Discounts). Features integrated transparency layers, custom image backgrounds, and responsive formatting settings.

### 2. Supplier Operations
Tailored for procurement teams to assess supplier health, geographical distribution maps, and isolated contact profiles. Built using advanced filter actions set to exclude values upon clearing for maximum database efficiency.
